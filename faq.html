I'm sure you have many questions, so let's go!

Can others see what I write?

Nope, no chance. Unless they find your key. Keep it safe! Create a new key if yours has been compromised.

Can you see what I write?

Nope! My code is on github for anyone to look at. It's a static website and only stores data locally on your pc, using localStorage (see below).

What's localStorage?

It's a safe way of storing data in your browser, that no one else can get access to (unless of course you have viruses...). Though as the name implies, data is locally stored, so if you want to use multiple devices, you need to set up your keys on all of them. But you only need to do this once (phew).

If his website gets hacked, can they take any sensitive data?

Nope, and even if it was hacked, they wouldn't find anything interesting. I'm running this website "statically", away from databases.

Eh?

There is nothing secretive about static websites. What you see if what you get. There's no "back-end", so there's no dodgy business going on. There are also no "cookies" (sad face) . Check out the code on github if you're still not convinced.
